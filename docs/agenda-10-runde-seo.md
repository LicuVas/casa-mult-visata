# AGENDĂ — 10 RUNDE SEO + GOOGLE ADS pentru casamultvisata.ro
Ghid de lucru pentru colaborarea Claude Web ⇄ Claude Code. Stabilit de Claude Code la cererea lui Vasile.
Scop final: plan executabil cap-la-coadă ca site-ul imobiliar local să capete tracțiune — SEO (organic, termen lung) + Google Ads (plătit, rapid). Fiecare rundă se închide cu CONSENS scris în `consens-seo-casamultvisata.md`.

## Regula fiecărei runde
1. Claude Web propune teza rundei (concret, nu generic).
2. Claude Code confirmă/corectează cu date reale (cartiere, CMS, constrângeri client) + adaugă garduri.
3. Consens → Claude Code îl scrie pe disc.
4. NIMIC ireversibil (build/deploy/ștergeri) fără OK explicit de la Vasile.

## Cele 10 runde (ordine logică: fundație → conținut → tehnic → trafic → măsurare → execuție)
- **R1 — Arhitectură site & intenția de căutare.** ✅ ÎNCHIS. Piloni tip×oraș + zone (nume corecte din nomenclator) + comune limitrofe separat; o intenție/pagină; URL curate; pagini de zonă progresiv; filtre ne-indexabile.
- **R2 — Cercetare cuvinte-cheie & intenții.** Lista seed reală de termeni (PN imobiliar, long-tail pe tip+zonă+camere), cum o obținem gratis (Keyword Planner/Trends/Autocomplete/SERP), gruparea pe intenție (informațional vs tranzacțional).
- **R3 — Conținut on-page (template pagină categorie/zonă).** H1/title/meta-description/FAQ/text unic per pilon și zonă; ce câmpuri vin din CMS vs ce e static; cum evităm conținut subțire.
- **R4 — SEO tehnic (Astro).** Sitemap, robots, canonical, viteză/Core Web Vitals, mobile-first, date structurate (Organization/LocalBusiness + RealEstateListing/Product), indexare.
- **R5 — SEO local & Google Business Profile.** GBP optimizat, NAP consistent, citations RO, strategie recenzii CONFORMĂ Google (fără gating, fără stimulente).
- **R6 — Conversie (CRO) & tracking conversii.** Ce transformă vizitatorul în lead: formular, click-telefon, click-WhatsApp, foto, CTA, semnale de încredere; definirea conversiilor ÎNAINTE de a plăti reclame.
- **R7 — Google Ads (campanie Search Leads).** Structură grupuri (terenuri/case/apartamente/general), keywords + match types, negative din start, licitare, buget (pe cifrele din `plan-google-ads-casamultvisata.md`).
- **R8 — Măsurare & decizie.** GA4 + Search Console + conversion tracking; KPI (CTR, CPC, rată conversie, cost-per-lead, calitate lead); pragul de decizie la 30 zile (scalezi/oprești/schimbi).
- **R9 — Concurență, diferențiere & conformitate legală.** Cine sunt agențiile/portalurile PN și ce fac; cum ne diferențiem; CONFORMITATE: poze cu persoane / drept la imagine pe site comercial (consimțământ), claim-uri corecte.
- **R10 — Plan de execuție & prioritizare.** Roadmap pe etape; ce face Vasile vs ce pot automatiza eu (CC); checklist go-live; ce necesită OK-ul clientului.
